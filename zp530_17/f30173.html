【スマート 4G WI-FI コンバータ】
ナノSIMカードスロット付け、4G LTE信号をWi-Fiネットワークに転送。
利用バンド指定可能。最大300Mbps(2.45GHz)Wi-Fi速度。
イーサネットが故障した場合、4G LTE無線ネットワークに変えて、データ伝送の続きを確保。

【オープンソース&プログラム可能】
「OpenWrt」を搭載されている、機能は拡張性があり、DIYプロジェクトに最適。
128MB RAM、16MB NOR + 128MB NANDフラッシュ、最大512MB MicroSDカード対応。
デュアルイーサネットポート、USB 2.0ポート、アンテナSMAマウントホールリザーブ。
さらに拡張し、デバイス間でファイルを共有するのに最適。

【セキュリティ&プライバシー】
OpenVPN & WireGuardが組み込まれていて、30+ VPN サービス プロバイダーをサポート。
GL.iNetが開発したWeb UI 3.0で、VPNサーバーとクライアントを簡単に構築。
IPv6、WPA3、Cloudfare対応。 オンラインのセキュリティをレベルアップ。

【Web UI 3.0 と GoodCloudで簡単設定】
GoodCloud はいつでもどこでもデバイスの管理と監視が可能になります。
リアルタイムで統計情報確認し、VPN サーバーとクライアントを設定し、クライアント接続リストを管理し、リモートSSH でIoT デバイスを管理できます。
内蔵されている4GモデムはATコマンド、手動/自動ダイヤル番号、SMSチェック、Web UI 3.0での信号強度チェックをサポートし、より優れた管理と設定を実現。
